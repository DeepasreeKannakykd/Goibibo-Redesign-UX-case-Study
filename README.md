## 🚀 Project Overview

This project was completed as my **major project during my UI/UX internship at Technglobal Pvt. Ltd.**, focusing on the redesign of the Goibibo website to improve usability, accessibility, and overall visual appeal.

The goal was to enhance the user experience of the travel booking platform by creating a cleaner, more intuitive, and user-friendly interface.

The redesign process included user research, identifying usability issues, wireframing, designing high-fidelity UI screens, and building an interactive prototype using Figma.

The final design delivers a modern, visually appealing, and easy-to-navigate travel booking experience.

---

## ❗ Problem in the Original Website

The original website had **navigation and page-state inconsistency issues**:

- Flight booking widget visible on Home page  
- Active indicator (blue line) remained under “Flights” tab incorrectly  
- Mismatch between:
  - Selected page  
  - Visible content  
  - Active navigation state  

This created confusion and reduced usability.

---

## 💡 Solution

To solve these issues, I redesigned the website using a **user-centered UI/UX approach**.

### ✅ Key Improvements

- Home page shows only relevant content  
- Flight booking widget appears only on Flights page  
- Active navigation indicator updates dynamically  
- Redesigned service pages:
  - Flights  
  - Cabs  
  - Buses  
  - Trains  
  - Hotels  
- Improved typography, spacing, and layout  
- Added relevant visuals and icons  

This redesign improves clarity, usability, and navigation experience.
